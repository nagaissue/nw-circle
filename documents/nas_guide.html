NAS（共有フォルダー）アクセスガイド

社内ネットワークに接続した状態で、以下の手順に従って共有ストレージ（NAS）にアクセスしてください。接続情報をお手元に用意して進めるとスムーズです。

ネットワークパス
\\10.2.39.2\network
ユーザー名
network
パスワード
network
ネットワークアクセス時の注意点:

環境や端末の状態により、最初のアドレス入力から接続画面が開くまでに数十秒程度の遅延（待ち時間）が発生する場合があります。エラー表示が出ない限り、そのまま閉じずにお待ちください。

1
Windows検索機能 ステップ 1 / 4

「エクスプローラー」を検索する

Windowsのデスクトップ画面下部（タスクバー）にある検索バー（または虫眼鏡アイコン）をクリックし、キーボードでエクスプローラーと入力します。

💻 Windows 画面イメージ 検索欄
エクスプローラー |
2
アプリ起動 ステップ 2 / 4

エクスプローラーを起動する

検索結果に表示されたフォルダーのアイコンをした「エクスプローラー」をクリックして開きます。

キーボードの Win ⊞ + E キーを同時に押すことでも、エクスプローラーを一瞬で起動できます。

アドレス入力 ステップ 3 / 4

アドレスバーにパスを入力

エクスプローラー上部にある「アドレスバー」の余白部分を一度クリックして入力状態にし、下記のパスを貼り付けて Enter キーを押します。

\\10.2.39.2\network

※ ¥マークと \ (バックスラッシュ)は同じ意味です。どちらで入力されても問題ありません。

認証情報の入力 ステップ 4 / 4

ユーザー名とパスワードの入力

接続確認ダイアログが表示されたら、以下情報を正しく入力し、最後に「OK」ボタンをクリックします。

入力する内容 どちらも共通です
ユーザー名 :
network
パスワード :
network
次回以降、入力を省略したい場合はダイアログ内の「資格情報を記憶する」にチェックを入れてから「OK」をクリックしてください。

補足：フォルダーの構成情報

無事に接続に成功すると、フォルダーの一覧が表示されます。主に、目的別の以下のような重要フォルダが用意されています。

"2025年度以降" フォルダ

用途：作成資料の保管

日々の作業で作成した成果物や提出用資料などを蓄積している親フォルダーです。

"最新パケトレ" フォルダ

用途：ツール用インストーラー

Cisco Packet Tracer（パケットトレーサー）の公式インストーラーを格納しています。新規導入・アップデート時にご活用ください。
