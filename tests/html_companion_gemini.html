HTML/CSS 体系的講義 対象：初学者 〜 中級者

サークルのホームページや資料を作成する際、まずは文書を形づくる「基本のタグ」を使いこなしましょう。左側のスライドもこれらのタグを組み合わせて作られています。

おまじない Line 1

<!DOCTYPE html>

ブラウザに対し、このファイルが最新規格の「HTML5」で記述されていることを宣言する非常に重要な1行です。これがないと、最新のCSS表示が崩れてしまう原因になります。 

<!DOCTYPE html>
コンテナ <div id="slideFrame">

セクション設計とID属性

スライド全体の枠組みは、汎用コンテナタグである <div> に id="slideFrame" を割り当てて作成しています。特定の1つだけの要素にスタイルを適用したり、JavaScriptで中身を操作したりする際に id を設定します。 

<div id="slideFrame" class="slide-ratio...">
  <!-- ここに中身 -->
</div>
ヘッダー構造 <div class="flex...">

階層構造としてのマークアップ

スライド上部の見出し部分は、見出しタグ <h2> と説明テキスト <p> が包含されています。見出しタグは1（重要度高）〜6（重要度低）までの数字があり、ページの文書構造を検索エンジンやブラウザへ伝える重要な役割を持ちます。 

<h2 class="text-xl font-bold...">ネットワークサークル</h2>
<p class="text-xs...">〜インフラからサーバー構築まで〜</p>
フッター <div class="mt-4...">

セマンティック（意味のある）マークアップ

スライドの最下部には著作権表記やスライド番号が配置されています。かつてはすべて <div> で済まされていましたが、現在は最下部を <footer>、最上部を <header> などの意味を持つタグで書くことが推奨されています。

ただタグを並べるだけでは、要素は縦一列に並ぶだけです。CSSを使って、パワーポイントのような高度なレイアウトを実現する仕組みを解説します。

CSS Grid grid-cols-12

12分割グリッドシステム

スライドは全体が縦1行・横12分割の「グリッド」で配置されています。 左コラムは12マスのうちの 「4マス分（col-span-4）」 を使っています。 

class="grid grid-cols-12 gap-4"
└ col-span-4 (左カラム)
└ col-span-5 (中央カラム)
└ col-span-3 (右カラム)
ボックスアライメント flex flex-col

Flexbox による縦並び配置

中央カラム内では、見出しやSVGマップが縦方向にきれいに並んでいます。これはCSSの display: flex; flex-direction: column; という指定をTailwindの flex flex-col を使って簡潔に記述しています。

アスペクト比 aspect-ratio

16:9比率の自動維持技術

スライドの形をいかなる端末の画面サイズでも「パワーポイントの1ページ」として崩さないために、CSS3の新機能である aspect-ratio: 16 / 9; を使用しています。幅がリサイズされると、高さが自動で算出されて追従します。

HTML/CSSで描いた静的な資料に、動きと機能（プログラム）をプラスするのがJavaScriptです。この資料に搭載した2つの仕組みを学びましょう。

トポロジー連動 onclick="selectNode(...)"

const nodeData = {

キーイベントの監視と条件分岐

入力枠 <input> へのキー入力を常時監視し、Enterキー（keyCode: 13）が押された瞬間に、入力文字列が "enable" か "configure terminal" かをIF文で判定し、プロンプト表示を変更しています。
