# 2026年度 ネットワークサークル活動計画

作成日：2026年4月26日
作成者：永井秀汰

## 1. 年間活動計画

## 2. 活動方針

サークル活動は主に作成資料（パワーポイント等）をもとに学習を進める．また，サークルメンバーのレベルに応じた柔軟な活動体制をとっていく．

サークル活動において独自の取組みとして挙げられるものを以下に示す．

- オリジナル教材の作成・共有・更新
- IT豆知識
