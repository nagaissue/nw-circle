## NASアクセスガイド

以下の手順に従ってNASにアクセスしてください。

:::message alert
**【重要】接続情報の取得について**

セキュリティ保護のため、ネットワークパス・ログイン名・パスワードは非公開となっています。
**「ネットワークパス」「ユーザー名」「パスワード」については、サークル長にお訊ねの上、取得してください。**
:::

| 項目 | 値 |
|---|---|
| ネットワークパス | `\\x.x.x.x\*******` |
| ユーザー名 | `*******` |
| パスワード | `*******` |

:::message
**ネットワークアクセス時の注意点**

環境や端末の状態により、最初のアドレス入力から接続画面が開くまでに**数十秒程度の遅延（待ち時間）**が発生する場合があります。エラー表示が出ない限り、そのまま閉じずにお待ちください。
:::

## 手順

### STEP 1/4：「エクスプローラー」を検索する

<sup>Windows検索機能</sup>

Windows のデスクトップ画面下部（タスクバー）にある**検索バー（または虫眼鏡アイコン）**をクリックし、キーボードで `エクスプローラー` と入力します。

> 💻 Windows 画面イメージ（検索欄）
> ```
> 🔍  エクスプローラー▏
> ```

### STEP 2/4：エクスプローラーを起動する

<sup>アプリ起動</sup>

検索結果に表示されたフォルダーのアイコンをした「**エクスプローラー**」をクリックして開きます。

:::message
キーボードの <kbd>Win ⊞</kbd> + <kbd>E</kbd> キーを同時に押すことでも起動できます。
:::

> 📂 アプリアイコン
> **エクスプローラー**（システム アプリ） — 開く（クリック）

### STEP 3/4：アドレスバーにパスを入力

<sup>アドレス入力</sup>

エクスプローラー上部にある「**アドレスバー**」の余白部分を一度クリックして入力状態にし、サークル長から取得したパスを入力して <kbd>Enter</kbd> キーを押します。

```
\\x.x.x.x\*******
```

※ ¥マークと \ (バックスラッシュ)は同じ意味です。どちらで入力されても問題ありません。

> 🖥️ エクスプローラー画面イメージ
> アドレスバー：`\\x.x.x.x\*******` ⏎ Enter
> 接続処理中...（数十秒かかる場合があります）

### STEP 4/4：ユーザー名とパスワードの入力

<sup>認証情報の入力</sup>

接続確認ダイアログが表示されたら、サークル長から取得したアカウント情報を正しく入力し、最後に「**OK**」ボタンをクリックします。

**入力内容**

| 項目 | 値 |
|---|---|
| ユーザー名 | `*******` |
| パスワード | `*******` |

:::message
次回以降、入力を省略したい場合はダイアログ内の「資格情報を記憶する」にチェックを入れてから「OK」をクリックしてください。
:::

> 🔐 Windows 資格情報の入力イメージ
>
> **Windows セキュリティ**
> ネットワーク資格情報の入力
> `\\x.x.x.x` に接続するための資格情報を入力してください。
>
> ユーザー名：`*******`
> パスワード：`••••••••`
> ☑ 資格情報を記憶する
>
> `[ OK ]` `[ キャンセル ]`

## 補足：フォルダーの構成情報

無事に接続に成功すると、フォルダーの一覧が表示されます。主に、目的別の以下のような重要フォルダが用意されています。

### 📁 "資料集" フォルダ

<sup>用途：作成資料の保管</sup>

日々の作業で作成した成果物や提出用資料などを蓄積している親フォルダーです。

### 📁 "CiscoPacketTracer_Installer" フォルダ

<sup>用途：ツール用インストーラー</sup>

Cisco Packet Tracer（パケットトレーサー）の公式インストーラーを格納しています。新規導入・アップデート時にご活用ください。
